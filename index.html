This demonstrates a little data binding in angular. The goal here is to make the hidden text show only when you check the checkbox, and also type in specifically checked or Checked in the text box!



The hidden h2 element has on it

ng-show="userInputCheck()



Which is a function that checks $scope.userInput to make sure the user has typed checked or Checked, AND that the checkbox is checked, and if so, returns true. userInput is bound to the text field (<input>) you can type in and has on it

ng-model="userInput"



The checkbox has

ng-model="checked"

on it which is set to false by default.



The console.log button just logs the value of userInput to the console for you to look at as you type and change it, etc etc.
